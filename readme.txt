Degc: Degree program requirements compiler

Requirements
============
- G++ 4.7 (or other compiler with similar C++11 support).
- GNU Make 
- Flex 2.5
- Bison 2.5
- Boost: http://boost.org
- NullUnit: http://code.google.com/p/nullunit

Build Instructions
==================
Run 'make' from the project directory.

$ make

Make will automatically build the project, run all test cases,
and place the output in ./bin
